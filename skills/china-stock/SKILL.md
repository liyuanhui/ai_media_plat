# 中国股票分析 Skill (增强版)

## 一、实时行情查询

### 腾讯财经 API (首选)
```bash
# 单只股票
curl -s "https://qt.gtimg.cn/q=sh600900" | iconv -f GBK -t UTF-8

# 多只股票批量查询 (推荐)
curl -s "https://qt.gtimg.cn/q=sh600900,sz002594,sh600522,sh601857,sh600036" | iconv -f GBK -t UTF-8
```

### 股票代码规则
- 上海股: `sh` + 6位数字 (如 sh600900)
- 深圳股: `sz` + 6位数字 (如 sz002594)
- 创业板: `sz` + 3开头 (如 sz300750)
- 北交所: `bj` + 8位数字

### 腾讯API字段说明
```
2=当前价 3=昨收 4=今开 5=成交量(手) 6=成交额(万) 7=最高 8=最低
13=市盈率 14=市净率 37=涨跌幅
```

## 二、热点新闻抓取

### 今日头条财经新闻 (推荐)
```bash
curl -s "https://www.toutiao.com/api/pc/feed/?category=stock&utm_source=toutiao&widen=1&max_behot_time=0" \
  -H "User-Agent: Mozilla/5.0" | python3 -c "
import sys,json
d=json.load(sys.stdin)
for i in d.get('data',[])[:15]:
    print(i.get('title',''))
"
```

### 新闻关键词过滤
搜索时使用以下关键词组合：
- 政策: "新能源政策"、"5G政策"、"AI政策"、"碳中和"、"国企改革"

## 三、股票推荐分析框架

### 推荐的5个维度

| 维度 | 权重 | 数据来源 |
|------|------|----------|
| **技术面** | 20% | 涨幅、量比、换手率 |
| **基本面** | 20% | 市盈率、市净器、ROE |
| **资金面** | 20% | 主力资金流向、成交量 |
| **政策面** | 25% | 行业政策、宏观政策 |
| **消息面** | 15% | 热点新闻、业绩预告 |

### 推荐股票筛选条件

**短线推荐** (5日内):
- 涨幅 > 3%
- 量比 > 1.5
- 换手率 > 3%
- 有利好新闻支撑

**长线推荐** (1-3月):
- 市盈率 < 25
- 市净率 < 5
- ROE > 10%
- 行业龙头

## 六、性能优化

1. **批量查询**: 一次API调用查5-10只股票
2. **curl+iconv**: 直接转码避免乱码
3. **新闻抓取**: 用今日头条API获取财经热点
4. **响应要快**: 不要铺垫，直接给数据和结论

## 七、注意事项

1. **风险提示**: 每次推荐必须包含风险提示
2. **不是投顾**: 强调仅供参考，不构成投资建议
3. **结合热点**: 每天分析要结合时事政策
4. **数据验证**: 先查行情再推荐，不能凭感觉
